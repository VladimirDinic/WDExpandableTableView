# Installation:
## Manual:
Download this project and add WDExpandableTableView folder (with WDExpandableTableView.swift file) to your project

## CocoaPods:
```Ruby
target '<TargetName>' do
    use_frameworks!
    pod 'WDExpandableTableView', ' ~> 0.0.9'
end
```
